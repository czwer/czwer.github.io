博客诞生记

博客搭建步骤：

  1. node安装：请下载6.0以下的node版本：
    • https://nodejs.org/dist/v5.5.0/
  2. 修改npm相关文件所有者：
    • sudo chown -R $(whoami) $(npm config get prefix)/{lib/node_modules,bin,share}
  3. 安装node npm源管理工具，并切换到taobao
    • npm install -g nrm
    • nrm use taobao
  4. 单独安装：dtrace-provider，避免后续下载卡主，参问题处理一
    • npm install -g dtrace-provider
  5. 安装 hexo-cli 并安装博客后，再安装npm，然后启动，本地输入：http://localhost:4000/，即可本地访问
    • npm install hexo-cli -g
    • hexo init blog
    • cd blog
    • npm install
    • hexo server
  6. 如果遇到DTraceProviderBindings MODULE_NOT_FOUND错误，参问题处理二
  7. gitHub上创建仓库：xxx.github.io，其中xxx为github账户
  8. 安装hexo-deployer-git：
    • npm install hexo-deployer-git –save
  9. 修改本地_config.yml中的deploy：
    • deploy:
      • type: git
      • repo: git@github.com:xxx/xxx.github.io.git
      • branch: master
      • message: 博客提交
  10. 创建博客：博客诞生记
    • hexo new 博客诞生记
  11. 生成静态文件：
    • hexo generate
  12. 发布到github：
    • hexo deploy
  13. 在github上进行外网访问：
    • https://xxx.github.io/

以上部分问题的处理,感谢泉兄的协助…
