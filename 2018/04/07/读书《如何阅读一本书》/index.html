读书《如何阅读一本书》

Apr 7, 2018 | Hits

由来：并发编程网推荐了几本书,其中有一本《如何阅读一本书》，自己也想提高下阅读质量与增进理解力。这几天读了一遍这本书，下面来是读书笔记：

一.读者的责任：

  • 关于一本书，一个阅读者要提出的四个基本问题：

    1. 整体来说，这本书到底在谈些什么？

      这是增进理解力的一本实用书。作为读者要做一名主动阅读的读者，为增进理解能力而阅读。阅读分为4个层次：基础阅读，检视阅读，分析阅读（重点介绍：读前->读中->读后），主题阅读。以及对不同阅读层次与不同类型读物的实践方法；最后讲了阅读的意义：增进理解力与心智。

    2. 作者细部说了什么，怎么说的？

      • 介绍阅读的层次：（阅读层次依次递增，带来的提升不同）
        • 基础阅读：字面意思；
        • 检视阅读：通读一遍，不求甚解；
        • 分析阅读：一本书从读厚到读薄，咀嚼与消化（重点介绍）；
        • 主题阅读：同类主题的书，比较阅读（经常在这里用到检视阅读），要能面面俱到，而自己不预设立场；
      • 提高阅读层次实现路径：
        • 读者：主动阅读，阅读越主动，效果越好（阅读是一种良好的自我教育方式）；
        • 书：读好书，且阅读对象理解力要高于读者，读者一定要把不相等的理解力克服到一定程度之内；
        • 方法论：通过一些规则指引；
      • 如何做一个自我要求的读者：
        • 一定要提出四个基本问题并回答；
        • 有做笔记的习惯；
        • 尽到读者应有的责任；
        • 培养阅读的习惯，熟练掌握持续增进；
      • 检视阅读有两种方式：阅读不同书籍/目的时，运用适当的不同速度
        • 有系统的略读或粗读：了解书的架构，初步判断是否是好书；
        • 粗浅的阅读：先啃一遍，了解全书内容；
      • 如何分析阅读：总结了指导实践的具体规则，并对不同类型读物，实践规则时进行适当的调整；
      • 进阶：进行更深层次的主题阅读，实践中理解力与心智的不断成长；
    3. 这本书说得有道理吗？是全部有道理，还是部分有道理？
      • 有道理。
    4. 这本书跟你有什么关系？
      • 运用书中的方法提高阅读质量与增进理解力。

二.如何检视阅读

  • 有系统的略读或粗读,具体做的建议（控制在1小时内）：
    1. 先看书名页，然后如果有序就先看序；
    2. 研究目录页；
    3. 如果书中附有索引，也要检阅一下；
    4. 读下出版者的介绍；
    5. 挑几个看来跟主题息息相关的篇章来看；
    6. 最后一步，把书打开，东翻翻西翻翻，念个一两段，有时候连续读几页，但不要太多；
  • 检视阅读时回答下面的问题：
    1. 这是什么样的一本书？
    2. 整本书在谈论的是什么？
    3. 作者是借着怎样的整体架构，来发展他的观点或陈述他对这个主题的理解？

三.如何分析阅读

  • 分析阅读的三阶段：

    • 第一个阶段：找出一本书在谈论什么的规则
      1. 依照书的种类与主题来分类
        • 论说类且实用性的书（有个基本面的判断）。
      2. 使用最简短的文字说明整本书在谈些什么？（整体性）
        • 掌握架构：介绍如何增进理解力的一本书，着重讲了分析阅读及不同类别读物的实践方法。
      3. 将主要部分按顺序与关联性列举出来。将全书的大纲列举出来，并将各个部分的大纲也列出来（复杂度）
        • 作者全书分成四个部分：
          • 第一部分谈的是：介绍阅读的层次（要干什么）；
          • 第二部分谈的是：如何分析阅读（一般怎么干）；
          • 第三部分谈的是：阅读不同读物的方法（具体怎么干）；
          • 第四部分谈的是：阅读的目标（进阶）；
        • 每部分的重点
          • 第一部分重点：认识到问题，指明努力的方向；
          • 第二部分重点：分析阅读的通用规则（指导怎么抓住重点，与作者沟通）；
          • 第三部分重点：针对不同类型读物，实践分析阅读规则时进行适当的调整；
          • 第四部分重点：进行更深层次的主题阅读，实践中理解力与心智的不断成长；
      4. 确定作者想要解决的问题
        • 作者希望提高读者阅读时的理解力，并针对分析阅读，总结了指导实践的具体规则，并针对不同类型读物，实践规则时进行适当的调整。
    • 第二个阶段：诠释一本书的内容规则
      1. 诠释作者的关键字，与他达成共识
        • 阅读，阅读层次，理解力，规则，沟通，语言，思想，知识，模糊，架构，主题，字义，启发，共识，主旨，规则，真理…
      2. 由最主要的句子中，抓住作者的重要主旨（问题的答案）
        • 列出的所有规则都是重要的句子。
      3. 知道作者的论述是什么，从内容中找出相关的句子，再重新架构出来
        • 对所有规则的解读。
      4. 确定作者已经解决了哪些问题，还有哪些是没解决的。再判断哪些是作者知道他没解决的问题
        • 解决的问题：如何抓住一本书的重点；
        • 未解决的问题：
          • 怎么判断读者正确地运用了作者的规则，没有误解其中的含义；
          • 如何系统化掌握一类知识，没有提供指导意见（还未扩展到面）；
        • 阅读本书目的：读者（思想/知识）<–>语言<–>书（文字）<–>语言<–>作者（思想/知识）：更好的沟通与理解；
    • 第三个阶段：像是沟通知识一样的评论一本书的规则（读后）
    • A.智慧礼节的一般规则
      1. 除非你已经完成大纲架构，也能诠释整本书了，否则不要轻易批评。（在你说出：“我读懂了！”之前，不要说你同意，不同意或暂缓评论。）；
        • 我读懂了，同意：认同分析阅读前检视阅读与个人实践一致（选择性接收）。
      2. 不要争强好胜，非辩到底不可（做人的道理）；
      3. 在说出评论之前，你要能证明自己区别得出真正的知识与个人观点的不同；

    • B.批评观点的特别标准：

      1. 证明作者的知识不足：
        • 出版年代原因，未提及计算机相关，虽然可以归类在科学与数学这一门类，期望后续版本，增加相关章节；
      2. 证明作者的知识错误；
      3. 证明作者不合逻辑；
      4. 证明作者的分析与理由是不完整的；

      注意：关于最后这四点，前三点是表示不同意见的准则，如果你无法提出相关的佐证，就必须同意作者的说法，或至少一部分说法。你只能因为最后一个理由，对本书暂缓评论。

四.阅读不同读物的方法

  • 划分以下几种读物类别
    1. 实用性的书：有道理的，落实行动；
    2. 想象文学：这类型读物传递体验（体验生命）
      • 故事：一口气通读后，再贯通；
      • 史诗：对作者与读者都是考验，毅力很重要；
      • 戏剧：更多解读的想象空间；
      • 悲剧：缺乏时间造成悲剧，道具暗示人物；
      • 抒情诗：一口气通读后，再贯通；
    3. 历史（电影《罗生门》比较形象）
      • 众人的历史：大概事实，比较阅读；
      • 个人的历史；
    4. 科学与数学：要有点数学基础很重要；
    5. 哲学：独立思考很重要；
    6. 社会科学：这类书可以理解为规范，建议广泛阅读；

五.如何主题阅读

  • 第一个阶段：观察研究范围：主题阅读的准备阶段：
    1. 针对你要研究的主题，设计一份试验性的书目。你可以参考图书馆目录，专家的建议与书中的书目索引；
    2. 浏览这份书目上所有的书，确定哪些与你的主题有关，并就你的主题建立起清楚的概念

  • 第二个阶段：主题阅读，阅读所有第一阶段收集到的书籍：

    1. 浏览所有在第一阶段被认定与你主题相关的书，找出最相关的章节；
    2. 根据主题创造一套中立的词汇，带引作者与你达成共识—无论作者是否实际用到这些词汇，所有的作者，或至少绝大部分的作者都可以用这套词汇来诠释；
    3. 建立一个中立的主旨，列出一连串的问题—无论作者是否明白谈过这些问题，所有的作者，或者至少大多数的作者都要能解读为针对这些问题提供了他们的回答；
    4. 界定主要及次要的议题。然后将作者针对各个问题的不同意见整理陈列在各个议题之旁。你要记住，各个作者之间或之中，不见得一定存在着某个议题。有时候，你需要针对一些不是作者主要关心范围的事情，把他的观点解读，才能构建出这种议题；

    5. 分析这些讨论。这得把问题和议题按顺序排列，以求突显主题。比较有共通性的议题，要放在比较没有共通性的议题之前。各个议题之间的关系也要清楚地界定出来；

      注意：理想上，要一直保持对话式的疏离与客观。要做到这一点，每当你要解读某个作家对一个议题的观点时，必须从他自己的文章中引一段话来并列出。

六.个人建议

  • 以下是一点点个人建议
    1. 读的时候，可以念出来，增加专注；
    2. 推荐用印象或有道做笔记，在笔记中可以记录下每章的重点，必要时用不同颜色区分；
    3. 几个人一起共同读一本书，互相学习；
    4. 找好书的方式：（a）可以从亚马逊上读同本书的其他人读的评分高的书；（b）在好书的封面看看推荐了那些书；（c）朋友推荐；
